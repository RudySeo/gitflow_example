# 1. :grin: 서비스 소개
* 칵테일 픽은 칵테일 정보 제공, 추천 서비스입니다. "좋은 술을, 고민 없이"라는 모토를 가지고 개발하고 있습니다.
---
<br>



# 3. :computer: 핵심 기능
* 칵테일 정보를 제공합니다.
    * 이름, 이미지, 어원, 특징, 도수, 맛, 레시피가 있습니다.
* 칵테일을 검색할 수 있습니다.
    * 칵테일 이름과 태그를 이용합니다.
* 칵테일을 추천합니다.
    * 사용자의 취향을 기반으로 합니다.
* 칵테일을 즐겨찾기할 수 있습니다.
